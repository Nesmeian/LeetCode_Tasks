# 219. Contains Duplicate II

**Difficulty:** Easy

## Problem Statement

Given an integer array `nums` and an integer `k`, return `true` if there are two distinct indices `i` and `j` in the array such that:

- `nums[i] == nums[j]`
- `abs(i - j) <= k`

## Examples

### Example 1

**Input:**

```plaintext
nums = [1, 2, 3, 1], k = 3
```
